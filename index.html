Later I studied C # programming in the .NET framework and database management with SQL Server, and now I have just completed a 500 hour bootcamp of web and mobile development (HTML5, CSS3, Javascript, Node.js, Typescript, Angular, Ionic, MySQL, MongoDB) at Codenotch (Madrid).

So I have a good understanding of the fundamentals of programming in general, and object-oriented programming in particular.
